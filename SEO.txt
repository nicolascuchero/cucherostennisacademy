1)PALABRAS CLAVE: agregué las siguientes keywords: "TENNIS" , "ACADEMY", "TRAINING".

2)AUTOR: agregué el meta name="author" con mi nombre: "NICOLAS CUCHERO"

La hice lo mas corta y consisa que pude asi queda bien clara y simple.

4)ALT: use alt en una foto con el siguiente texto: "intro".

5)ENCABEZADO: mediante el h1 utilicé el siguiente texto: "CUCHEROS TENNIS ACADEMY". 

6)TITULO: usé de title el siguiente texto: "Home". Es claro y corto para que el usuario sepa donde está ubicado.

7)TEXTOS: utilicé textos cortos y claros para que el usuario sepa qué está mirando y así lo comprenda mas rápido y fácil.

8) RESPONSIVE: todo el sitio está hecho con un diseño responsive para usar en distintos dispositivos, ya sean mobile o desktop.

9) VINCULOS: usé vinculos o links cuando puse algun video o alguna imagen. Básicamente un enlace al sitio web.

10) NAVEGACIÓN: para que el usuario se ubique bien en la página y puedan encontrar rapidamente el contenido que estan buscando, en la pagina de inicio pueden navegar desde ahi por todo el sitio web mediante un menú de navegación.
